##项目介绍
量纲分析库

###对象类型
|  中文  | 类名  | 说明  | 举例 |
| :----: | :---: | :--- | :--- |
| 量纲    |   Dims  | 量纲数组，一些标记 | $A^xB^yC^z$ |
| 物理量纲 | PhyDims | 量纲数组长度固定为7 | $MLT^{-2}$ |
| 物理单位 | PhyUnit | 保存量纲，名称，单位符号，与国际单位的倍率 | $N$ |
| 物理量   | PhyVar  | 保存符号，实际意义描述 | $F=?N$ |
| 物理值   | PhyVal  | 保存"物理量"和具体数值(可以是数字，也可以是数组) | $F=1N$ |


###计划
打印信息  
数量级词头  
解析字符串单位  
保存数据到sqlite
